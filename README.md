## How to start?
1. Clone the Repo.
2. run ```npm install``` at root dir.
3. run ```npm install``` at ```/frontend/login```
4. run ```npm start``` at the root dir. (This script should start the 3 apps at the same time.)

### Ports:
* ```3000``` Back-end
* ```5520``` Chat App
* ```4200``` Login App
